# Resume App.

SPA for creating, and storing resumes. Synchronizing data with the cloud.

* esponsive design.
* RUD operations.
* orm validation.
* eview and save to pdf.
* atabase authentication.
* toring data in a real-time database.

## Technologies:
* Javascript.
* React.
* MaterialUI.
* Firebase.
* @react-pdf/renderer.
