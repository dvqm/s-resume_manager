# Resume Manager.

SPA for creating, and storing resumes. Synchronizing data with the cloud.

* Responsive design.
* CRUD operations.
* Form validation.
* Review and save to pdf.
* Database authentication.
* Storing data in a real-time database.

## Technologies:
* Javascript.
* React.
* Material UI.
* Firebase.
* @react-pdf/renderer.
